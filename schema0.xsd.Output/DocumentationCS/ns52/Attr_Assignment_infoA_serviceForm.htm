ns52.ServiceForm ServiceForm
  Property Description  
    Property Name ServiceForm  
    Property Type ns52.ServiceForm  
    Parent Class ns52.Assignment_info  
    XML Type Element  
    Accessors Read/Write  
    Description An optional ServiceForm  
    Remarks It is optional, initially it is null.
In order to populate it a new ns52.ServiceForm object must be created and assigned to the property.
 
obj.ServiceForm = new ns52.ServiceForm();
 
To 'un-populate' the property set it to null.

 
    Example 
ns52.Assignment_info oParent = ...;

// The object is optional so initially is empty
System.Diagnostics.Debug.Assert(oParent.ServiceForm == null);
oParent.ServiceForm = new ns52.ServiceForm();
oParent.ServiceForm.[Some Property] = ...;
