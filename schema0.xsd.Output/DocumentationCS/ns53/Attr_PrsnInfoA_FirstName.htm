String FirstName
  Property Description  
    Property Name FirstName  
    Property Type String  
    Parent Class ns53.PrsnInfo  
    XML Type Element  
    Accessors Read/Write  
    Description A mandatory String  
    Remarks It is mandatory and therefore must be populated within the XML.
It is defaulted to [""].

 
    Example 
ns53.PrsnInfo oParent = ...;

// Because the property is mandatory, it will already be populated
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine(oParent.FirstName);

// And set as follows 
oParent.FirstName = "Sample Value";
