LiquidTechnologies.Runtime.Net40.XmlDateTime BirthDate
  Property Description  
    Property Name BirthDate  
    Property Type LiquidTechnologies.Runtime.Net40.XmlDateTime  
    Parent Class schemaEgisso.MSZ_receiver  
    XML Type Element  
    Accessors Read/Write  
    Description A mandatory LiquidTechnologies.Runtime.Net40.XmlDateTime  
    Remarks It is mandatory and therefore must be populated within the XML.
It is defaulted to [new LiquidTechnologies.Runtime.Net40.XmlDateTime(LiquidTechnologies.Runtime.Net40.XmlDateTime.DateType.date)].

 
    Example 
schemaEgisso.MSZ_receiver oParent = ...;

// Because the property is mandatory, it will already be populated
System.Diagnostics.Debug.Assert(oParent.BirthDate != null);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine( oParent.BirthDate.ToString();

// So it can be set as follows 
oParent.BirthDate.SetDate(2005, 11, 29);
      
// Or like this
LiquidTechnologies.Runtime.Net40.XmlDateTime oNewItem = new LiquidTechnologies.Runtime.Net40.XmlDateTime();
oNewItem.SetDate(2005, 11, 29);
oParent.BirthDate = oNewItem;
