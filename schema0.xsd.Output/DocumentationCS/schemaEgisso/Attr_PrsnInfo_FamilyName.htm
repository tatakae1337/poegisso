String FamilyName
  Property Description  
    Property Name FamilyName  
    Property Type String  
    Parent Class schemaEgisso.PrsnInfo  
    XML Type Element  
    Accessors Read/Write  
    Description A mandatory String  
    Remarks It is mandatory and therefore must be populated within the XML.
It is defaulted to [""].

 
    Example 
schemaEgisso.PrsnInfo oParent = ...;

// Because the property is mandatory, it will already be populated
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine(oParent.FamilyName);

// And set as follows 
oParent.FamilyName = "Sample Value";
