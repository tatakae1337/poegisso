LiquidTechnologies.Runtime.Net40.XmlDateTime LastChanging
  Property Description  
    Property Name LastChanging  
    Property Type LiquidTechnologies.Runtime.Net40.XmlDateTime  
    Parent Class ns51.Fact  
    XML Type Element  
    Accessors Read/Write  
    Description A mandatory LiquidTechnologies.Runtime.Net40.XmlDateTime  
    Remarks It is mandatory and therefore must be populated within the XML.
It is defaulted to [new LiquidTechnologies.Runtime.Net40.XmlDateTime(LiquidTechnologies.Runtime.Net40.XmlDateTime.DateType.dateTime)].

 
    Example 
ns51.Fact oParent = ...;

// Because the property is mandatory, it will already be populated
System.Diagnostics.Debug.Assert(oParent.LastChanging != null);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine( oParent.LastChanging.ToString();

// So it can be set as follows 
oParent.LastChanging.SetDateTime(2005, 11, 29, 12, 56, 36);
      
// Or like this
LiquidTechnologies.Runtime.Net40.XmlDateTime oNewItem = new LiquidTechnologies.Runtime.Net40.XmlDateTime();
oNewItem.SetDateTime(2005, 11, 29, 12, 56, 36);
oParent.LastChanging = oNewItem;
